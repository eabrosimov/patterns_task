Паттерн "Прототип"

В данном примере создается фабрика монстров, которая содержит
внутри себя объект типа Monster, в котором реализован метод copy()
интерфейса Copyable()
Этот метод способен создавать новый объект типа Monster и присваивать
ему такие же значени для полей как и у оригинала.
Фабрика возвращает копию монстра приводя его к типу конкретной реализации,
чтобы избежать приведение типов в классе клиенте
