Паттерн "Мост"

В данном примере абстрактный класс Character характеризует игрового персонажа.
У него есть реализации: лучник, маг, воин.
Интерфейс Weapon характеризует оружие, которое имеет свои особенности при нанесении
урона. Имеется 3 реализации: лук, посох, молот

Каждый персонаж потенциально может сражаться любым оружием. Чтобы постоянно не менять
классы наследники Character при добавлении нового оружия и его характеристик абстракция
Weapon отделяется от реализаций абстрактного класса Character. Таким образом изменения в
Character и Weapon никак не повлияют друг на друга.
