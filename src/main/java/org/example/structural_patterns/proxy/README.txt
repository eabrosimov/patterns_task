Паттерн "Прокси"

В данном примере с помощью интерфейса Game
можно присвоить объект типа прокси, которые
перехватит управления основным объектом и
будет управлять вызовами его методов

Работая с обычным объектом при выборе игрового
сервера происходила автоматическая авторизация
даже, если пользователь не планировал играть.
С помощью прокси пользователь сначала выбирает
сервер, а потом уже решает заходить на него или нет
