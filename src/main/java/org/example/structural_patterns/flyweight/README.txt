Паттерн "Приспособленец"

В данном примере классы Dragon, Wolf, Skeleton
характеризуют монстров на карте.
Монстры одного типа выглядят совершенно одинаково,
у них меняется лишь точка респавна исходя из координат.
Поэтому для отрисовки каждого из монстров можно использовать
один и тот же объект класса отрисовки.
Создается фабрика по отрисовке, в которую добавляется объект для
отрисовки монстра лишь один раз. Далее используется один и тот же объект
