Паттерн "Хранитель"

В данном примере реализован класс Character, у которого есть поле item типа String.
Для того, чтобы сохранить значение этого поля используется объект типа Warehouse,
для которого так же определено поле item. Значение переменной item персонажа присваивается
переменной item у Warehouse. Сам объект Warehouse помещается в объект типа WarehouseKeeper.
