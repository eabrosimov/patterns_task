Паттерн "Наблюдатель"

В данном примере реализованы классы от интерфейса Observer и Observed.
Класс типа Observed является наблюдаемым объектом, который сообщает об изменении своего
состояния классам типа Observer, которые являются наблюдателями и реагируют на все изменения
наблюдаемого объекта.

В класса StockMarker определена переменная indexSnP500, которая характеризует цену индекса
на рынке ценных бумаг. Так же определен лист наблюдателей. Как только цена индекса меняется,
все наблюдатели уведомляются с помощью метода notifyObserver(), который вызывает у каждого
наблюдателя метод handleEvent() и передает в параметры измененную цену индекса. В зависимости
от цены, каждый наблюдатель обрабатывает событие по своему.
