Паттерн "Посредник"

В данном примере класс MyChat реализующий интерфейс Chat и его метод sendMessage().
Также класс SimpleUser реализующий интерфейс User и его метода sendMessage() и getMessage().

Чат содержит лист пользователей и является посредником. Пользователь отправляет сообщение вызывая метод
sendMessage() у переменной типа Chat. Метод реализован так, что сообщение от пользователя попадает в цикл
и с помощью метода getMessage() отправляется всем пользователем содержащимся в листе кроме того, который отправил сообщение.
